# AI-Lab Roadmap

AI-Lab is a modular research environment for long-term AI-assisted scientific work, with RTG as the first primary research domain.

## v0.0.x — Foundation

- Working Python project structure
- Protected secrets
- GitHub integration
- Tests
- README
- Engineering Constitution
- Provider abstraction
- OpenAI provider

## v0.1.x — Usable AI Core

- Stable provider interface
- Conversation manager
- Prompt templates
- Logging
- Basic history
- CLI entry point

## v0.2.x — RTG Research Layer

- DSN loader
- DSN version tracking
- DSN section indexing
- Context pack builder
- Evidence-level tagging:
  - certified
  - conjectural
  - exploratory
  - stale
  - superseded

## v0.3.x — Scientific Artifact System

- Experiment output archive
- Result manifests
- Reproducibility metadata
- Generated report storage
- Figure/table archive
- Git-linked scientific artifacts

## v0.4.x — Scientific Tool Integration

- Numerical libraries
- Symbolic math libraries
- Graph/network libraries
- Notebook integration
- AI/ML research libraries
- External model/provider integrations

## v0.5.x — Persistent Research Memory

- Local project memory
- Session summaries
- Research state snapshots
- Decision history
- DSN-linked memory retrieval
- Long-term RTG continuity layer

## v0.6.x — Multi-Agent Discussion

- Open discussion between agents
- Planner / critic / verifier roles
- Cross-model review
- Debate mode
- Consensus and disagreement reports
- Human approval gates

## Long-Term Vision

AI-Lab should become a research instrument, not only a chatbot wrapper.

It should help preserve:
- code,
- scientific artifacts,
- DSNs,
- decisions,
- experiments,
- discussions,
- failures,
- and the reasoning behind them.

The goal is continuity: future researchers, developers, or AI assistants should be able to continue the work without reconstructing the past from memory.
